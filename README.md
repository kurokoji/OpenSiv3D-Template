# OpenSiv3D-Template

[OpenSiv3D-Template](https://github.com/kurokoji/OpenSiv3D-Template) is a template for cross-platform development of [OpenSiv3D](https://github.com/Siv3D/OpenSiv3D) using CMake.

## Support Platform

- Windows []
- macOS [x]
- Linux []

## Requirements

- CMake >= 3.12.0

## Usage

### Setup

```bash
./setup.sh
```

### Build

```bash
mkdir build
cmake ..
make
```
